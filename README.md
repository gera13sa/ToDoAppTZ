# ToDo

iOS-приложение для управления задачами, написанное на **SwiftUI** с архитектурой **VIPER**.

При первом запуске задачи загружаются из [DummyJSON](https://dummyjson.com/docs/todos), сохраняются локально и дальше работают без повторного запроса к API.

## Возможности

- Список задач с названием, описанием и датой
- Поиск по названию и описанию
- Добавление, редактирование и удаление задач
- Отметка задачи как выполненной
- Изменение даты задачи
- Контекстное меню: редактировать, поделиться, удалить

## Стек

- SwiftUI
- VIPER
- `URLSession` + `Codable`
- Локальное хранение в CoreData

## Требования

- Xcode 16+
- iOS 17+

## Запуск

1. Клонируйте репозиторий
2. Откройте `ToDo.xcodeproj` в Xcode
3. Выберите симулятор или устройство
4. Нажмите **Run** (⌘R)

> Для первой загрузки задач нужен интернет. Чтобы снова загрузить данные с API, удалите приложение с устройства и установите заново.
